JSON数据增强功能测试页面

此页面用于测试JSON数据增强功能，包括动态数据库下拉列表、表下拉列表、表选择JSON生成和关系JSON查看功能。

新增功能说明

1. 动态数据库下拉列表

从表元数据JSON文件中自动提取数据库信息，动态生成数据库下拉列表选项

2. 动态表下拉列表

根据选择的数据库，从表元数据中过滤并显示对应的表列表

3. 表选择JSON生成

通过表选择功能，生成包含选中表完整结构信息的JSON文件

4. 关系JSON查看

提供查看当前表关系JSON数据的功能，支持在新窗口中预览
